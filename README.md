# JDBC-Movie-Booking-System

A desktop-based Movie Booking System developed using **Java Swing** and **JDBC** for database connectivity. This project allows users to log in, select movies, choose show timings, book seats, and manage movie ticket reservations through an interactive graphical user interface.

---

# 📌 Project Overview

The JDBC Movie Booking System is designed to simulate an online movie ticket booking platform. The application provides a smooth and user-friendly interface where users can:

* Login to the system
* Select preferred language
* Browse available movies
* Choose show timings
* Select seats
* Confirm ticket booking
* Connect with MySQL database using JDBC

This project demonstrates the practical implementation of:

* Java Swing GUI Development
* JDBC Database Connectivity
* Event Handling
* Object-Oriented Programming Concepts
* MySQL Database Operations

# ⚙️ How It Works

## 1️⃣ Login Page

Users first log in to the system.

## 2️⃣ Dashboard

After successful login, the dashboard is displayed.

## 3️⃣ Language Selection

Users can choose their preferred movie language.

## 4️⃣ Movie Selection

Available movies are displayed with posters and details.

## 5️⃣ Time Selection

# 💻 Installation & Setup

## Step 1: Clone Repository

```bash
git clone https://github.com/your-username/JDBC-Movie-Booking-System.git
```

## Step 2: Open Project

Open the project in:

* VS Code
* Eclipse
* IntelliJ IDEA

## Step 3: Add MySQL Connector

Add the JDBC MySQL Connector Jar file:

```bash
mysql-connector-j-9.6.0.jar
```

## Step 4: Setup Database

Run the SQL queries provided in:

```bash
JDBC SQL Code.txt
```

## Step 5: Configure Database Credentials

Update database username and password inside:

```java
DBConnection.java
```

## Step 6: Run the Project

Execute:

```bash
Login.java
```
